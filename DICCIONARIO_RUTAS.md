# Diccionario Geográfico --- Mi Ruta

## Objetivo

Este documento define una convención única para nombrar sectores, puntos
de interés y referencias geográficas utilizadas por **Mi Ruta**. El
propósito es utilizar códigos cortos, consistentes y fáciles de recordar
para rutas, paraderos, búsquedas y documentación.

------------------------------------------------------------------------

# Convenciones

## Código base

Cada sector o lugar utiliza un código base de **2 a 4 caracteres**,
procurando que sea fácilmente identificable.

Ejemplos:

  Lugar        Código
  ------------ --------
  Centro       CT
  Mapasingue   MAP
  Alborada     ALB
  Guasmo       GUA
  Kennedy      KEN

## Puntos cardinales

Cuando un sector posee divisiones geográficas se agrega un sufijo.

  Sufijo   Significado
  -------- -------------
  N        Norte
  S        Sur
  E        Este
  O        Oeste

Ejemplos:

-   MAPE = Mapasingue Este
-   MAPO = Mapasingue Oeste
-   KENN = Kennedy Norte
-   GUAS = Guasmo Sur

## Subdivisiones numeradas

Las **etapas, bloques, cooperativas, sectores, manzanas o cualquier
división numerada** utilizan exactamente la misma regla: **Código base +
número**.

Ejemplos:

-   ALB1 = Alborada Etapa 1
-   ALB5 = Alborada Etapa 5
-   SAU6 = Sauces 6
-   BAS4 = Bastión Bloque 4
-   BAS10 = Bastión Bloque 10

## Subdivisiones especiales

Cuando no existe numeración ni punto cardinal:

  Sufijo   Significado
  -------- -------------
  C        Central
  V        Vieja
  A        Alta
  B        Baja

Ejemplos:

-   URDC = Urdesa Central
-   KENV = Kennedy Vieja

# Categorías

  Prefijo           Categoría
  ----------------- -----------------------------------------------
  *(Sin prefijo)*   Sectores, barrios, ciudadelas, urbanizaciones
  V                 Red vial (vías, avenidas, calles, corredores)
  T                 Terminales
  M                 Centros comerciales
  U                 Universidades
  C                 Colegios
  H                 Hospitales y clínicas
  P                 Parques
  R                 Mercados y puntos de referencia

# Reglas finales

1.  Los códigos deben ser cortos y fáciles de recordar.
2.  Deben derivarse del nombre del lugar siempre que sea posible.
3.  Los sectores no utilizan prefijo.
4.  Los puntos de interés utilizan el prefijo de su categoría.
5.  Los puntos cardinales utilizan N, S, E y O.
6.  Todas las subdivisiones numeradas utilizan la regla **Código +
    Número**.
7.  Evitar crear excepciones salvo para resolver duplicados.
